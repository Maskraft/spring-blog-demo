[[overview]]
== 概要

SpringBlog が提供する REST API の仕様書です。このドキュメントは https://spring.io/projects/spring-restdocs[Spring REST Docs] を用い、`ArticleApiDocumentation` テストの実行結果から自動生成されています。

[[overview-base-url]]
=== ベース URL

すべてのエンドポイントは以下のベース URL 配下に存在します。

[cols="1,3"]
|===
| 項目 | 値

| バージョン
| `v1`

| ベース URL
| `\http://localhost:8080/api/v1`
|===

[[overview-content-type]]
=== Content-Type

リクエスト・レスポンスともに JSON（UTF-8）を使用します。

- リクエスト: `Content-Type: application/json`
- レスポンス: `Content-Type: application/json;charset=UTF-8`

[[overview-errors]]
=== エラーレスポンス形式

エラー発生時は以下の形式で JSON を返します。

[cols="2,1,4"]
|===
| フィールド | 型 | 説明

| `timestamp` | String | エラー発生日時（ISO-8601）
| `status`    | Number | HTTP ステータスコード
| `error`     | String | HTTP ステータス名
| `message`   | String | エラーメッセージ（バリデーション時はフィールド毎の違反を `;` で連結）
|===
